# PopX UI App

A modern and responsive React.js application that implements a complete user onboarding flow including Welcome, Signup, Login, and Account Settings pages.

## Features
- Clean and minimal UI design
- Fully responsive layout (mobile, tablet, desktop)
- User onboarding flow:
  - Welcome Page
  - Create Account
  - Login Page
  - Account Settings
- Reusable layout structure
- Styled using Tailwind CSS
- Client-side routing using React Router

## Tech Stack
- React.js
- Tailwind CSS
- React Router
- Vite

##  Pages
- Home (Welcome)
- Create Account
- Login
- Account Settings

## Deployment
Deployed on Vercel with proper routing configuration.

## 📌Note
This project focuses on frontend UI implementation and does not include backend authentication.
